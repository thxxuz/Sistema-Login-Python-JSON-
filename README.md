# Sistema de Login em Python (JSON)

Sistema básico de cadastro e login utilizando arquivos JSON para armazenar usuários.

## Funcionalidades
- Criar nova conta
- Fazer login
- Armazenamento dos dados em JSON
- Criação automática do arquivo se ele não existir

## Tecnologias
- Python 3
- Biblioteca nativa "json"

## Como executar
1. Clone o repositório:
   git clone https://github.com/thxxuz/sistema-login-python

2. Entre no diretório:
   cd sistema-login-python

3. Execute:
   python main.py

## Estrutura do projeto (exemplo)
main.py        → código principal  
funcoes.py     → funções do sistema  
usuarios.json  → arquivo onde ficam os dados  

## Observações
- Para resetar o sistema, basta apagar o arquivo usuarios.json.
